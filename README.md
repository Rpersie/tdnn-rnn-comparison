# PT-BR Análise comparativa  de RNNs e TDNNs aplicadas ao reconhecimento de fala

Este repositório armazena os modelos experimentais bem como os resultados do Trabalho de Conclusão de Curso de [Lucas Valente](https://www.github.com/serramatutu) e [Felipe Martins](https://www.github.com/Fyvon0), desenvolvido para o curso de Informática Diurno do Colégio Técnico de Campinas, COTUCA, durante o período de fevereiro a novembro de 2018.
